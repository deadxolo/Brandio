Quick Start.

Copy · Paste · Ship

List your brands

Every workspace lives in its own brand. Pull the full list to start wiring things up.

curl http://localhost:3004/api/businesses \ -H "Content-Type: application/json"

Create a new brand

Spin up a new isolated workspace — assets, schedules, and analytics all scoped to it.

curl -X POST http://localhost:3004/api/businesses \ -H "Content-Type: application/json" \ -d '{"name":"Acme Co","industry":"SaaS"}'

Check service health

Each service exposes /api/health. Useful for uptime checks and CI.

curl http://localhost:3004/api/health

Get raw service map

Need the original JSON for tooling? It's still at the root API path.

curl http://localhost:3004/api | jq
